# DRF-PROJECT: Muscle Mall
### DRF의 Serializer, Permission을 연습하고 CBV를 사용한 API 작성, TDD 및 CI & CD를 공부하는 프로젝트 입니다. 
<hr>

## 소개
`MM Project` 은 내게 맞는 운동, 식단, 영양 루틴을 구성하거나, 각 분야(운동, 식단, 영양)의 코치들의 루틴을 참고하거나 해당 코치의 지도를 받을 수 있는 서비스 입니다.
