# DRF-PROJECT: Muscle Mall 

## 소개
`MM Project` 은 내게 맞는 `Product`(운동, 식단, 영양 루틴)을 구성하거나, 각 분야(운동, 식단, 영양)의 코치들의 `Challenge`를 참고하거나 해당 코치의 지도를 받을 수 있는 서비스 입니다.  <br>
